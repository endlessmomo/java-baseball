## 💻우아한 테크 프리코스 2주차 (야구 게임)

## 기능 구현
- 컴퓨터
    - [ ] 중복되지 않는 세자리 난수를 생성 기능

- 사용자 입력
    - [ ] 게임 진행 여부 및 종료
    - [ ]숫자 입력 기능 ( 난수와 비교하기 위한 )

- 데이터 검증
    - [ ] 입력 에러
        - 숫자가 아닌 다른 문자가 입력되는 경우
        - 조건에 벗어나는 입력값
        - 각 자리수가 중복되게 입력된 경우

- 결과
  - [ ] 사용자가 입력한 수와 생성된 난수의 비교를 통해 스트라이크/볼/낫싱 판정 기능
